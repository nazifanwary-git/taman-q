# Taman-Q

A website for those interested in starting their own hydroponic garden and learn more about the types of plants they can grow.

## Live Demo
https://taman-q.vercel.app/

## Tech Stack
- Lucide React
- React
- React Router
- Tailwind CSS
- Vite

## Features
- Responsive design for mobile and desktop
- (Hopefully) Easy to follow quick start guide with illustrations

## What I Learned
- Design and create reusable components
- Implement site navigation with React Router
